# SudokuPuzzleGenerator
By Aliya Saldanha-Suri

This program generates a fully solved 9x9 Sudoku puzzle! This was created using Java code learned in the AP Computer Science A course.

Run Sudoku.java in a Java IDE and it will generate a Sudoku board.

The Sudoku board is generated by first generating the three diagonal boxes of the board, and then systematically generating each row, resetting the puzzle if there are no valid row options. It can generate a puzzle in ~2 seconds consistently with simple methods!

This project includes the Sudoku.java source code, the design document, and the README file.
